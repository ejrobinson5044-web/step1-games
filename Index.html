HOW THIS WORKS

Each game saves progress to localStorage in the browser you're using. That means:

  • Saves are per-device, per-browser. Your phone and laptop track progress separately.
  • Use the 🔄 SYNC button in any game's menu to copy a save code between devices.
  • Don't use Incognito/Private mode — saves get wiped when the tab closes.
  • Bookmark this library page instead of individual games. One bookmark, seven games.
